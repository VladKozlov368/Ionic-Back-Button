# Ionic-Back-Button

Overrides the Android back button completely

## Getting Started

### Install Cordova
```
cordova plugin add pro.inmost.ionic.back.button
```

### Install Ionic
```
ionic cordova plugin add pro.inmost.ionic.back.button
```

## Example
```
document.addEventListener('onBackButtonExt', (ev) => {
  // code...
});
```
